# 🏮 Китай за 10 дней

Одностраничный сайт-путеводитель для самостоятельного путешествия по Китаю со стартом в Шанхае.

**Что умеет:**
- 📅 Параметры поездки: даты приезда/отъезда, город прилёта и вылета, число путешественников —
  длина маршрута и бюджет пересчитываются автоматически
- 🧭 Конструктор маршрута: выбираете города — сайт сам распределяет дни,
  строит порядок посещения (от города прилёта к городу вылета) и показывает,
  как добраться между городами (поезд/самолёт, время, цена)
- 📍 Детальные планы по дням: достопримечательности с временем, ценами и советами
- 💳 Памятка туриста: транспорт, деньги (Alipay/WeChat Pay), интернет и VPN, еда, виза
- 🗣 Мини-разговорник с русской транскрипцией
- 📱 Полностью адаптивен под мобильные

## Структура

```
index.html        — вся страница
css/style.css     — стили (светлая тема, красные акценты)
js/data.js        — база городов, достопримечательностей и переездов
js/app.js         — логика конструктора маршрута
images/           — фотографии (см. images/PHOTOS.md — какие нужны)
```

Никаких сборщиков и зависимостей — чистые HTML/CSS/JS.

## Как выложить на GitHub Pages

1. Создайте репозиторий на github.com (например, `china-trip`)
2. Загрузите туда все файлы проекта (через веб-интерфейс «Add file → Upload files» или git)
3. В репозитории: **Settings → Pages → Source: Deploy from a branch → Branch: main / (root) → Save**
4. Через пару минут сайт будет доступен по адресу
   `https://<ваш-логин>.github.io/china-trip/`

## Как добавить фотографии

Список нужных файлов и где их бесплатно скачать — в [images/PHOTOS.md](images/PHOTOS.md).
Без фотографий сайт тоже работает: вместо них показываются цветные заглушки.

## Как редактировать маршрут

Все города и достопримечательности — в `js/data.js`. Чтобы добавить город, скопируйте
любой объект в массиве `CITIES`, поменяйте данные и добавьте переезды в `TRANSFERS`.
