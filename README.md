# 🏮 Китай — конструктор путешествия

Одностраничный сайт-путеводитель для самостоятельного путешествия по Китаю.

**Что умеет:**
- 📅 Параметры поездки: даты приезда/отъезда, город прилёта и вылета, число путешественников —
  длина маршрута и бюджет пересчитываются автоматически
- 🧭 Конструктор маршрута: выбираете города — сайт сам распределяет дни, строит
  **оптимальный порядок посещения** (по цене и времени переездов, от города прилёта
  к городу вылета) и показывает, как добраться между городами (поезд/самолёт, время, цена)
- 📍 Детальные планы по дням: достопримечательности с временем, ценами и советами
- 💳 Памятка туриста: транспорт, деньги (Alipay/WeChat Pay), интернет и VPN, еда, виза
- 🗣 Мини-разговорник с русской транскрипцией
- 📱 Полностью адаптивен под мобильные, оформление — liquid glass поверх фото-фона

## Структура

```
index.html        — вся страница
css/style.css     — стили (liquid glass, красный/золотой акценты)
js/data.js        — база городов, достопримечательностей и переездов
js/app.js         — логика конструктора маршрута и оптимизация порядка городов
images/           — фотографии городов и фон сайта
```
